Privacy Policy

Regarding data collection

ECyD Culiacán by itself does not collect any personal data from the user. All personal data is used only to facilitate tournament enrollment, and isn't tied to an specific person.

Regarding the use of the User ID

ECyD Culiacán doesn't make use of the User ID but the integrated Firebase SDK does, only for basic analytics data.
